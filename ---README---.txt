Pentru instalare:
- descarca si instaleaza nodejs: https://nodejs.org/en/download
- deschide cmd cu administrator in folderul asta si scrie: npm install

-> Poti modifica shortcutul pentru a deschide text boxul de mesaje in main.js (clickThroughShortcut)
-> Daca vrei sa stilizezi chatul adauga CSS in user-style.css

-> Deschide iar cmd cu admin in folderul asta si scrie: npm run pack
-> O sa ai apara un folder numit dist, in interior gasesti executabilul Wap-Forum-Overlay.exe (poti sa faci un shortcut si sa-l pui pe desktop)
-> Deschide-l si apasa combinatia de taste pe care ai setat-o sau Control+Shift+F10 daca nu ai setat nimic.
-> Login cu datele tale
-> O sa iti apara fereastra WF cu fundal alb, da click pe fereastra WF si ALT+F4 si deschide din nou app

-> Daca vrei sa faci orice modificare este nevoie sa executi din nou comanda de build (npm run pack) pentru a adauga modificarile tale in executabil.
